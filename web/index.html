How it works

Torpedo uses LSB (Least Significant Bit) steganography to hide data in the least significant bits of image pixels. This makes the changes virtually invisible to the human eye.

  • Supports PNG, JPEG, WebP, BMP and other formats
  • Optional XOR encryption with password
  • Capacity: ~1 byte per 8 pixels
  • Uses browser Canvas API for format conversion
